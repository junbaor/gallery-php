# gallery-php

> 练习 php 时写的图床, 使用七牛云存储  

### 注意  
敏感信息未删除, 但已失效.

### 使用
- 修改 `config.php` 中 myslq 数据库信息
- 修改 `save.php`  中七牛云 accessKey、secretKey 以及图片的域名
